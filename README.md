# Rock-Paper-Scissors

## Description:
 This is an interactive, browser based rock, paper and scissors game that lets you play against a robot opponent. it has a clean interface, hand shaking animation, & score tracking, etc

## Why i built this:
 To level up my front-end skills by building a project from scratch. it made me practice state management in Javacript, and manipulate DOM without relying on frameworks.

## Tech Stack:
 * HTML: Core Structure of the game.
 * CSS: custom variables & keyframe animations.
 * Javascript: Game logic, event listeners, etc.

## How it works:
 When you click a button *rock, paper or scissors* the games locks the response to prevent spam or triggers. once the cooldown ends, the script checks a random choice for bot, swaps the emojis to reveal hands, and check the outcomes against a victory lookup table to update the scoreboard and check for the winnner.
